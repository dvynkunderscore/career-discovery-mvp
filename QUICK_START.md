# 🚀 Quick Start Guide

## One-Minute Setup

### Local Development
```bash
# 1. Clone the repo
git clone https://github.com/dvynkunderscore/career-discovery-mvp.git
cd career-discovery-mvp

# 2. Install dependencies
npm install

# 3. Start dev server
npm run dev

# 4. Open browser
open http://localhost:3000
```

### Deploy to Vercel (Recommended - 1 Click)

1. Go to https://vercel.com/new
2. Select this repository
3. Click "Deploy"
4. Done! Your app is live ✨

### Deploy to Netlify

```bash
# Build locally
npm run build

# Deploy the .next folder to Netlify
```

## 📋 What to Test

### Screens
- ✅ **Screen 1**: Fill in age, gender, interests
- ✅ **Screen 2**: Make 5 binary choices (Tinder-style)
- ✅ **Screen 3**: Watch the gradient number reveal
- ✅ **Screen 4**: Choose "Modify" or "Proceed"
- ✅ **Screen 5**: Scroll through careers by category
- ✅ **Screen 6**: Explore career details (day-in-life, skills, courses, projects)

### Design System
- ✅ Pure monochrome (white background, black text)
- ✅ Retro monospace fonts display correctly
- ✅ Typewriter effect adds periods after words
- ✅ Gradient numbers only (vibrant colors)
- ✅ Smooth animations and transitions
- ✅ Responsive on mobile, tablet, desktop

## 🐛 Troubleshooting

**Port 3000 already in use?**
```bash
npm run dev -- -p 3001
```

**Dependencies issues?**
```bash
rm -rf node_modules package-lock.json
npm install
```

**Fonts not loading?**
Clear browser cache and restart dev server
